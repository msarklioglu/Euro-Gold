# 🏆 Altın Takip — Kurulum

## Gerekenler
- Node.js (https://nodejs.org)

## Kurulum

### 1. İlk seferinde (bir kez yapılır):
Klasörü aç, terminali aç ve şunu yaz:
```
npm install
```

### 2. Her seferinde çalıştırmak için:
```
node server.js
```

### 3. Tarayıcıda aç:
```
http://localhost:3000
```

## Notlar
- Fiyatlar hakanaltin.com'dan gerçek zamanlı geliyor
- Bilgisayarı kapattığında tekrar `node server.js` çalıştır
